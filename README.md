
# Pokedex App

Pokedex App is a web-based Single Page Application (SPA) that allows users to explore the world of Pok√©mon using the Master-Detail design pattern. The app features a paginated list of Pok√©mon, allowing users to navigate through the vast array of creatures in the Pok√©mon universe.

## Tech Stack

**Framework:** [NextJS](https://nextjs.org/),TypeScript

**Data Fetching:** [SWR](https://swr.vercel.app/es-ES), Fetch

**Styles:** [TailwindCSS](https://tailwindcss.com/), [shadcn/ui](https://ui.shadcn.com/)

**Api:** [Pok√©mon API](https://pokeapi.co/)
## Installation

To get started with the Pokedex app, you will need to install the necessary Node modules. To do this, follow these steps:

```bash
  cd poke-app
  npm i 

```

## Getting Started

First, run the development server:

```bash
npm run dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


## Running Tests

To run tests, run the following command

```bash
  npm run test
```
